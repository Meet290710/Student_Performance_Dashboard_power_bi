# рҹ“Ң Project Overview

The **Student Performance Dashboard** is an interactive Power BI report designed to analyze students' academic performance, attendance, and classroom behavior.

The dashboard enables teachers, school administrators, and educational institutions to monitor student performance across different classes, subjects, and academic terms while identifying trends that support better decision-making.

# рҹҺҜ Objectives

* Analyze student academic performance.
* Track attendance percentage.
* Monitor classroom behavior.
* Compare performance across subjects and classes.
* Identify high, medium, and low performers.
* Build an interactive dashboard using Power BI.

# рҹ’Ў Key Insights

### рҹ“ҳ Academic Insights

* Mathematics achieved the highest average score.
* Student performance improved from Term 1 to Term 3.
* Most students fall into the **Medium Performance** category.
* A small group of students consistently achieved scores above 80%.

### рҹ“… Attendance Insights

* Overall attendance remained above 90%.
* Students with higher attendance generally scored better academically.
* Absences were concentrated among a limited number of students.

### рҹҳҠ Behavior Insights

* Positive behavior incidents significantly outnumbered negative incidents.
* Students with repeated negative behavior records generally had lower academic performance.
* Most classrooms demonstrated positive behavioral trends.
